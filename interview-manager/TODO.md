# 面接事前テスト管理 残タスク

## 完了: Supabase Auth + RLS

2026-07-20に本番適用・公開・権限確認済み。

- GROP管理者とBARAENは株式会社オオタ面接を取得できる
- AKANE、VJC、未ログイン状態では株式会社オオタ面接を取得できない
- 未ログイン状態では面接、候補者、クレペリン結果を直接取得できない
- QR受験用のベトナム国語・クレペリン保存関数は未ログイン状態から呼び出せる
- 株式会社オオタ面接の送り出しはBARAEN

## 残タスク

- 新しい送り出しを追加する場合は、DB制約・ログイン選択肢・RLSアカウントを同時に更新する

## 完了: 送り出しスタッフ画面の整理

- PDF・CSVボタンはGROP管理者だけに表示
- 候補者削除ボタンはGROP管理者だけに表示
- 候補者削除はRLSでもGROP管理者だけに許可

## 完了: ピンボード評価の4段階化

- 各回をプルダウンで `◎ 正確に完了`、`○ 自力で修正`、`△ 一部ミス`、`× 継続困難` から選択
- ◎・○の場合だけ完了時間を入力
- 2回分の評価合計、2回目の評価、完了時間の順で順位付け
- 管理画面に4段階の判断基準と順位ルールを常時表示
- 入力欄は「ピンボード 1回目評価・時間」「2回目評価・時間」と表記
- 印刷用集計表とCSVにも各回の評価・時間を出力
