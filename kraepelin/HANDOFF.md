## 🧮 検査仕様

- 1行 116個の1桁数字（1〜9のランダム）
- 隣り合う2数字を足して**一の位**を入力
- 1行60秒、**前半15行 → 休憩5分 → 後半15行**（計30行・約35分）
- 入力: テンキーUI + キーボード（0〜9）

## 🖥 画面構成

### スタート画面
- 検査の説明・例題ボックス
- 「練習する（1行）」「検査開始」ボタン

### 検査画面
- ヘッダー: 前半/後半バッジ、行数、タイマー（残り10秒で赤）
- 数字列（横スクロール、現在の2数字をハイライト）
- 回答セル（正誤を緑/赤で即時表示）
- テンキー（モバイル対応、touchstartで即反応）

### 休憩画面
- 5分カウントダウン、スキップボタン

### 結果画面（PDF出力対応）
1. **総合判定カード** — 定型 / 準定型 / 非定型
2. **作業曲線グラフ** — Canvas描画、前半青/後半橙、休憩区切り、エリア塗りつぶし
3. **4軸評価**（性格・行動特性、5段階バー表示）
   - 発動性（取りかかりの良さ）
   - 可変性（気分・行動の変化）
   - 亢進性（勢いの強さ）
   - 作業量（処理能力・効率）
4. **作業能力スコア** — 総回答数・正答数・誤答数・正答率・1行平均
5. **詳細分析** — 初頭努力 / 疲労耐性 / 休憩効果 / 前半vs後半 / 正確性
6. **各行の詳細テーブル** — 最高/最低行ハイライト、前半/後半バッジ
7. **PDF保存 / もう一度** ボタン

## 🧠 判定ロジック（results.js）

### 定型 / 非定型 判定（100点満点・減点方式）
- 平均作業量 < 20問/行 → -30点
- 誤答率 > 15% → -20点
- 初頭努力なし（最初3行 ≤ 中盤平均）→ -15点
- 休憩後の回復なし → -15点
- 標準偏差 > 8（変動大） → -20点
- 途中で50%以上の急落 → -15点

判定: `≥70点=定型` / `40-69点=準定型` / `<40点=非定型`

### 4軸評価（5段階）
| 軸 | 計算方法 |
|---|---|
| 発動性 | 最初3行平均 / 全体平均 の比率 |
| 可変性 | 全体の標準偏差 |
| 亢進性 | 休憩後の回復量 + 後半持続度 |
| 作業量 | 1行平均正答数（50/40/30/20が境目） |

## 🖨 PDF出力仕様（A4 2枚）

- `window.print()` を使用（外部ライブラリなし）
- `@page { size: A4; margin: 12mm 14mm; }`
- フォント・余白・グラフ高さ（180px）を縮小
- スコアカードを横5列に
- 各セクションに `break-inside: avoid` を指定

## ⚠️ 注意点

- `index.html`, `style.css`, `app.js`, `results.js` の4ファイルだけで完結
- 外部ライブラリは使っていないので、追加する場合はCDNリンクで対応推奨
- Canvas描画は `devicePixelRatio` 対応済み（高解像度ディスプレイOK）
- モバイル対応済み（375px幅から動作）
- 練習モードは1行だけ実行できる（テスト用）

## 🐛 既知の動作仕様

- 検査画面で物理キーボードの数字キーを押せば入力可能（0〜9）
- 行を全て解き終わると自動で次の行へ（時間切れを待たない）
- 休憩はスキップ可能
- 結果画面では `Results.render(allResults)` を直接呼べばモックデータでも表示可能
